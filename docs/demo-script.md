# Demo Script Outline

## Target Length

3 to 5 minutes

## Narrative Arc

1. Introduce the portfolio and the decision-support goal.
2. Show the portfolio construction workflow.
3. Run a historical stress event and explain the loss path.
4. Switch to a hypothetical macro shock and compare the estimated impact.
5. Close with recommendations, explainability, and interview takeaways.
